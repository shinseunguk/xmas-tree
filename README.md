# 🎄 Christmas Tree Animation

눈이 내리는 크리스마스 트리 애니메이션 프로그램입니다.

## ✨ 기능

- **ASCII 아트 크리스마스 트리**: 별과 함께하는 아름다운 크리스마스 트리
- **반짝이는 효과**: 개별 문자가 노란색으로 랜덤하게 반짝임
- **눈 내리는 애니메이션**: 다양한 모양의 눈송이가 떨어짐
- **컬러풀한 디스플레이**: ANSI 색상을 이용한 터미널 출력
- **갈색 나무 기둥**: 자연스러운 트리 모양

## 🚀 실행 방법

```bash
python3 xmas_tree.py
```

## 🎮 사용법

- 프로그램을 실행하면 자동으로 애니메이션이 시작됩니다
- `Ctrl+C`를 눌러 프로그램을 종료할 수 있습니다

## 🛠️ 요구사항

- Python 3.x
- 터미널에서 ANSI 색상 지원

## 📝 코드 구조

- `ChristmasTree` 클래스: 메인 애니메이션 클래스
- `Colors` 클래스: ANSI 색상 코드 정의
- `generate_tree()`: 트리 구조 생성
- `generate_snow()`: 눈송이 생성 및 움직임
- `render_frame()`: 화면 렌더링
- `animate()`: 메인 애니메이션 루프

## 🎄 특징

### 트리 구조
- 3단계 섹션으로 구성된 자연스러운 트리 모양
- 상단에 별(★) 장식
- 갈색 나무 기둥

### 눈 애니메이션
- 다양한 눈송이 모양 (`*`, `.`, `o`, `~`)
- 좌우 흔들림 효과
- 화면 경계 내에서 자연스러운 움직임

### 색상 효과
- 초록색 기본 트리
- 10% 확률로 노란색 반짝임
- 갈색 나무 기둥
- 다양한 색상의 눈송이
